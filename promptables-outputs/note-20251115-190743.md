AI coder App Prompt: Pura Growth
I want to build a comprehensive marketing automation platform that helps our company manage multi-platform social media campaigns, educational email courses, and giveaway automation for supplement brands. It should feel professional and corporate and be optimized for web desktop use.

Project Name:
Pura Growth

Target Audience:
• Internal company team members managing supplement brand campaigns
• Content creators and marketers within our organization
• Campaign analysts tracking performance metrics

Core Features and Pages:

Dashboard Homepage
• Unified overview displaying campaign performance metrics, content calendar preview, active giveaway campaigns, email course analytics, and recent social media performance
• Quick action buttons for creating new campaigns, scheduling content, and viewing reports
• Real-time notifications for keyword triggers, campaign milestones, and performance alerts

Content Management System
• Content calendar with drag-and-drop scheduling for 30 daily posts across Instagram, TikTok, and Twitter
• Template library for storing and organizing reusable content formats
• AI-powered content generation using OpenAI integration for captions, hashtags, and post ideas
• Media asset manager with Supabase storage for images and videos
• Content performance tracking with engagement metrics, reach, and conversion data
• Bulk upload and batch scheduling capabilities

Giveaway Campaign Manager
• Campaign creation wizard for setting up 8-12 monthly giveaway campaigns
• Keyword trigger system for "Comment [KEYWORD] to get [lead magnet]" workflows
• Automated response tracking and lead capture integration
• Campaign performance analytics including participation rates, conversion metrics, and ROI tracking
• Template library for common giveaway formats and copy

Educational Email Course Builder
• 8-day email sequence builder with drag-and-drop email editor
• Landing page creator for each EEC with conversion-optimized templates
• Klaviyo integration for automated email delivery and subscriber management
• A/B testing capabilities for email subject lines and content
• Performance tracking for opt-in rates, open rates, click-through rates, and conversions
• Customer research tools for identifying pain points and creating targeted courses

Social Media Integration Hub
• Native API connections to Instagram, Twitter, and TikTok for posting and analytics
• Cross-platform content adaptation tools
• Automated posting schedules with platform-specific optimization
• Social listening for keyword mentions and engagement opportunities
• Direct message automation for lead capture workflows

Analytics and Reporting Center
• Comprehensive performance dashboard with follower growth, engagement rates, and email list growth
• ROI tracking for campaigns and email courses
• Custom report builder for stakeholder presentations
• Goal tracking and milestone monitoring
• Export capabilities for data analysis

Tech Stack (Recommended Defaults):
• Frontend: Next.js 14, Tailwind CSS, shadcn/UI
• Backend & Storage: Supabase
• CMS: Built-in content management system
• Auth: Email/password login with team collaboration features
• Integrations: OpenAI API, Klaviyo API, Instagram API, Twitter API, TikTok API, Gemini API, VEO3 API

Design Preferences:
• Font: Inter
• Colors:
   - Primary: #22c55e (Light Green)
   - Accent: #15803d (Dark Green)
   - Backgrounds: clean white, soft grays
• Layout: Desktop-first, professional corporate design with clean navigation and data visualization

AI-Powered Features:
• Content generation using OpenAI for social media posts, email sequences, and campaign copy
• Image and video generation using Gemini and VEO3 APIs for visual content creation
• Smart scheduling recommendations based on audience engagement patterns
• Automated hashtag and caption suggestions for maximum reach
