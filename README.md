# Pura Growth

A comprehensive marketing automation platform for supplement brands, designed to streamline multi-platform social media campaigns, educational email courses, and giveaway automation.

## Features

- **Dashboard**: Real-time performance metrics and campaign overview
- **Content Management**: Calendar-based scheduling with AI-powered content generation
- **Campaign Management**: Giveaway campaigns with keyword triggers and automation
- **Email Courses**: 8-day educational sequences with landing pages
- **Social Media Integration**: Instagram, TikTok, and Twitter management
- **Analytics**: Comprehensive performance tracking and reporting
- **Integrations**: Third-party service connections (Klaviyo, OpenAI, etc.)
- **Settings**: User preferences and system configuration

## Tech Stack

- **Frontend**: React 18 with Vite
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Date Handling**: date-fns

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
